# Лабораторна робота №4

## 4.1 Однорідна система

dx/dt = P(x, y)  
dy/dt = Q(x, y)

P & Q - однорідні функції одного степеня  

## 4.2 Лінійна система

dx/dt = ax + by  
dy/dt = cx + dy

## Приклади для систем

    P(x,y) = x^3 - 3xy^2  
    Q(x,y) = 3x^2y - y^3
